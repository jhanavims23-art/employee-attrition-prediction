# Employee Attrition Prediction using Machine Learning
 Project Overview

This project focuses on analyzing employee data and predicting attrition risk using Machine Learning techniques. The goal is to help organizations identify employees who are likely to leave and take proactive retention measures.



## 🎯 Project Objective

* Analyze employee attrition trends
* Identify key factors influencing attrition
* Build a predictive model to estimate attrition probability
* Provide actionable HR insights

---

## 📊 Dataset Information

* Total Employees: 1470
* Features: 35
* Key variables include:

  * Age
  * Department
  * Monthly Income
  * Job Satisfaction
  * Work-Life Balance
  * Overtime
  * Attrition (Yes/No)

---

## 🔍 Key Findings

* Overall Attrition Rate: **16.12%**
* Sales department has highest attrition (**20.63%**)
* Employees working overtime have significantly higher attrition
* Lower salary employees are more likely to leave
* Poor work-life balance increases attrition risk

---

## 🤖 Machine Learning Model

* Model Used: Logistic Regression
* Data Split: 80% Training, 20% Testing
* Model Accuracy: **84.69%**

---

## 🔮 Predictive Insights

* HR has slightly higher average attrition probability
* Sales has highest concentration of high-risk employees
* R&D shows relatively stable attrition
